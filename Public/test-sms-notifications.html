Ethiopia Telecom SMS Setup Guide
Step 1: Contact Ethiopia Telecom

Contact Ethiopia Telecom's business services to request SMS API access:

  • Phone: +251-11-515-5555
  • Email: business@ethiotelecom.et
  • Visit: Local Ethiopia Telecom office
Step 2: Required Information

You'll need to provide:

  • Business registration documents
  • Purpose of SMS usage (customer notifications)
  • Expected monthly SMS volume
  • Technical contact information
Step 3: API Credentials

Once approved, you'll receive:

  • API Key
  • Sender ID (e.g., "SARAHS")
  • API endpoint URL
  • Documentation
